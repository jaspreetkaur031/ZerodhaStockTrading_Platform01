# 📈 Zerodha Stock Trading Platform

A full-stack stock trading platform inspired by Zerodha, built using the MERN Stack.

## 🚀 Features

- User Authentication
- Holdings Management
- Positions Tracking
- Order Management
- Portfolio Dashboard
- Interactive Charts
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Material UI
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Charts
- Chart.js

## 📂 Project Structure

```bash
├── frontend
├── backend
├── dashboard
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/jaspreetkaur031/ZerodhaStockTrading_Platform01.git
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm run dev
```
